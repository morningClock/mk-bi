# mk-bi 可视化大屏项目

MK-bi是一个基于Vue3+Echarts+ts的可视化大屏项目，主要涵盖了基础的可视化图形用例。

目标是涵盖工作中出现的所有大屏应用的用例。

如果此项目对你有用，欢迎star!

## 技术栈

1.  vue3
2.  typescript
3.  Tailwind
4.  echarts
5.  three.js
6.  node.js

## 项目说明

```bash
# 安装依赖
pnpm i
# 本地项目
npm run dev
```
